# 📚 WikiSearch
A modern and responsive Wikipedia Search Application built using **HTML, CSS, JavaScript, Bootstrap, and Fetch API**. Search Wikipedia articles in real time with a clean, user-friendly interface.

## 🚀 Live Demo
https://harshityadav-tech.github.io/Wikipedia-search-app/


## ✨ Features
- 🔍 Real-time Wikipedia article search
- 🌐 Fetches live data using Wikipedia Search API
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and dynamic search experience
- ⏳ Loading spinner while fetching results
- 🔗 Clickable article titles and URLs
- 🎨 Clean and modern UI
- 💻 Built using Vanilla JavaScript
---
## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4
- Fetch API
---



## 🎯 How It Works

1. Enter a keyword in the search box.
2. Press **Enter**.
3. The application sends a request to the Wikipedia Search API.
4. Search results are displayed dynamically.
5. Click any title to open the full Wikipedia article.
